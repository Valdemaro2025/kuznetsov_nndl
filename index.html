üí° Key Insight: The Most Important Factor of Death on Titanic

After analyzing the Titanic dataset, the most significant factor affecting survival was passenger class (Pclass).

First-class passengers had a survival rate of approximately 63%, compared to 47% for second-class and only 24% for third-class passengers.

This stark disparity highlights the socioeconomic hierarchy aboard the Titanic, where first-class passengers had priority access to lifeboats and were located closer to the boat deck.

Other important factors included:

  • Sex: Women had significantly higher survival rates (74%) than men (19%)
  • Age: Children had higher survival rates than adults
  • Fare: Higher fare-paying passengers had better survival chances

The "women and children first" protocol was followed, but class privilege ultimately played a decisive role in determining who survived the disaster.

This app runs entirely in the browser with no backend required. To deploy it on GitHub Pages:

  1. Create a public GitHub repository
  2. Commit index.html and app.js to the main branch
  3. Go to Repository Settings ‚Üí Pages ‚Üí Select "main" branch as source
  4. Save and wait a minute, then open the generated URL
